# My Portfolio
<p align="justify">This website displays projects, work experience, and contact information.</p>

This repository is a generic portfolio template. Replace the contents of `index.html`, `style.css`, and `main.js` with your own assets and information.

## Sections
- Mini intro
- About
- Skills
- Projects
- Contact

## Installation & Deployment
- Clone the repository and update `index.html` to customize the site.
- Replace images in the `src/` directory with your own assets.
- Update analytics and search console IDs in the `<head>` section if you use them.
